Unsupervised Learning Analysis
Dataset sintético generado automáticamente.
Incluye K-Means, estandarización y análisis de clusters.
Los clusters obtenidos muestran diferencias claras entre segmentos de clientes en variables relevantes 
como ingreso, monto del crédito y número de atrasos. Esto podría complementar el modelo supervisado 
permitiendo detectar subpoblaciones con riesgo diferenciado o validar la consistencia del scoring.
